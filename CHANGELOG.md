# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2025-11-17

Patch release: dependency alignment and maintenance.

### Fixed

- Updated dependency pins and internal housekeeping to align with other JetLeaf packages.

## [1.0.0] - 2025-11-17

Initial release.

### Added

- A set of high-level utilities and helpers used across JetLeaf applications (extensions, helpers, small adapters).
- Integration helpers for `jetleaf_lang` and `jetleaf_logging`.

### Notes

- This package provides utility primitives intended to be stable; for breaking changes we will document migration steps in future releases.
